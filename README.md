## CustomBlocksM
CustomBlocksM is a fork of the plugin [CustomBlocks](https://github.com/MMonkeyKiller/CustomBlocks), but with more abilities and more comfortable to use.

## Features:
* 1.17.X support
* Custom sounds
* Custom drop
* Tools supports
* Separated readable configs
* Columns
* Directional blocks [WIP]

## Dependencies:
* [LibraryM](https://www.spigotmc.org/resources/librarym.96514/)
* [NBTAPI](https://www.spigotmc.org/resources/nbt-api.7939/)
